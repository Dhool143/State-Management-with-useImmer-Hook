# User Profile with useImmer (React)

## 📌 Project Overview
This project demonstrates how to manage **complex nested state in React** using the `useImmer` hook.

Instead of using traditional `useState` with spread operators, this project uses **Immer** to simplify updates to deeply nested objects like user profiles.

---

## 🎯 Features
- Manage nested state using `useImmer`
- Update user details (name, email, phone, address)
- Toggle user preferences (newsletter, notifications)
- Real-time state display using `<pre>`
- Clean and responsive UI

---

## 🛠️ Technologies Used
- React (Vite)
- use-immer
- JavaScript (ES6)
- CSS
